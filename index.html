App Upgrading
Step-by-Step

From case to case the upgrading process differs, but there are common
stages you should know about.

01

Upgrade Planning

Before doing any upgrading activities we perform app environment inventory and review its compatibility so that we can create an upgrading plan.
02

Preparation to the Upgrade Process

Before performing upgrading itself we should check your apps first and undeploy them if necessary. We also back up the app environment at this stage
03

Upgrade Process

At this very step, we move your app to the newest frameworks’ versions, upgrade libraries, and set up APIs.
04

Post-Upgrade Procedures

The process is not finished, as we still need to verify an updated app, check the overall infrastructure, and only after that promote the app to production.
